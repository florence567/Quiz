# Interactive Quiz App

## Project Description

The Interactive Quiz App is a simple web-based application developed using HTML, CSS, and JavaScript. The application displays multiple-choice questions one at a time, allows users to select answers, calculates the final score, and displays the result at the end of the quiz.

This project is designed for beginners to understand the basics of frontend web development and JavaScript logic.

---

The main objective of this project is to learn and implement:

* HTML page structure
* CSS styling
* JavaScript interactivity
* DOM manipulation
* Event handling
* Conditional statements
* Arrays and objects
* Dynamic content updating

# Features

* Displays one question at a time
* Multiple-choice answer buttons
* "Next" button to move between questions
* Automatic score calculation
* Final result screen

# How the Application Works

1. The quiz starts by displaying the first question.
2. The user selects one answer from the available options.
3. The application checks whether the selected answer is correct.
4. The score increases for every correct answer.
5. The user clicks the "Next" button to continue.
6. After all questions are completed, the final score is displayed.
7. The user can restart the quiz using the restart button.
